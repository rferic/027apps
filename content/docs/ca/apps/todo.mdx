Tasques és un gestor de tasques senzill per a grups. Cada usuari pot crear, completar i eliminar les seves pròpies tasques. Els administradors poden configurar límits per usuari.

## Funcionalitats

- **Crear tasques**: Afegeix tasques amb un títol
- **Completar tasques**: Marca tasques com a fetes amb un sol clic
- **Límits per usuari**: Els administradors poden establir un màxim de tasques per usuari
- **Visibilitat**: Les tasques poden ser públiques o privades
- **Widget**: Widget al tauler amb tasques pendents i completades

## Configuration

Admins can configure the maximum number of tasks per user in the admin panel.

| Key | Type | Default | Min | Max |
|-----|------|---------|-----|-----|
| `max_items` | number | 50 | 1 | 1000 |
