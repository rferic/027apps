Inspiration lets group members submit ideas, report bugs, request features, and vote on what matters most. Every request goes through a status lifecycle: `pending → reviewing → approved → in_progress → completed`.

## Features

- **6 request types**: Bug, Improvement, New App, App Feature, General Functionality, Other
- **Voting**: Members can support ideas they like
- **Comments**: Discuss ideas with threaded comments
- **Status workflow**: Track ideas from submission to completion
- **Email notifications**: Admins get notified on new ideas and comments
- **Widget**: Dashboard widget showing most supported and recently completed
- **Changelog**: View completed ideas as a changelog
