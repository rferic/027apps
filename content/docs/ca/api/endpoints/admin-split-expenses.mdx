# Admin — Split Expenses

## Llistar tots els grups de despeses

```bash
GET /api/v1/admin/apps/split-expenses
```

Retorna tots els grups de despeses de tots els grups amb el nombre de membres i imports totals. Requereix rol `admin`.

**Auth:** JWT (`admin` role)

**Resposta 200:**
```json
[
  {
    "id": "uuid",
    "group_id": "uuid",
    "title": "Home Expenses",
    "emoji": "🏠",
    "currency": "EUR",
    "created_by": "uuid",
    "created_at": "2026-01-15T10:30:00Z",
    "member_count": 4,
    "total_amount": 1250.50
  }
]
```

## Estadístiques globals

```bash
GET /api/v1/admin/apps/split-expenses/stats
```

Retorna estadístiques agregades de tots els grups. Requereix rol `admin`.

**Auth:** JWT (`admin` role)

**Resposta 200:**
```json
{
  "total_groups": 3,
  "total_expenses": 156,
  "total_amount": 4520.75,
  "total_members": 12,
  "by_currency": {
    "EUR": { "groups": 2, "total": 3200.00 },
    "USD": { "groups": 1, "total": 1320.75 }
  }
}
```
