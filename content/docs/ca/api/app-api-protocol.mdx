# Protocol API per Apps

Les apps de l'ecosistema 027Apps poden exposar els seus propis endpoints REST. La plataforma gestiona automàticament l'enrutament, l'autenticació i el middleware — només cal que declaris quins endpoints exposa la teva app i on trobar els fitxers handler.

## Com declarar endpoints REST

### 1. Habilita API al manifest

Estableix `"api": true` al fitxer `manifest.json` de la teva app:

```json
{
  "id": "notes",
  "name": "Notes",
  "version": "1.0.0",
  "api": true
}
```

### 2. Crea fitxers de ruta

Col·loca els teus handlers de ruta a `apps/{slug}/routes/`. L'estructura de fitxers determina el camí URL:

```
apps/{slug}/routes/
├── route.ts           → GET   /api/v1/admin/apps/{slug}
├── [id]/
│   └── route.ts       → GET   /api/v1/admin/apps/{slug}/:id
│                       → PUT   /api/v1/admin/apps/{slug}/:id
│                       → PATCH /api/v1/admin/apps/{slug}/:id
│                       → DELETE /api/v1/admin/apps/{slug}/:id
```

### 3. Exporta mètodes HTTP

Cada fitxer de ruta exporta un o més de: `GET`, `POST`, `PUT`, `PATCH`, `DELETE`.

```typescript
// apps/notes/routes/route.ts
export async function GET(req: Request, ctx: UseCaseContext) {
  // ...
}

export async function POST(req: Request, ctx: UseCaseContext) {
  // ...
}
```

## Nivells d'autenticació estàndard

| Nivell | Capçalera | Cas d'ús |
|---|---|---|
| `public` | Cap | Dades no sensibles, compartides del grup |
| `jwt` | `Authorization: Bearer <token>` | Qualsevol usuari autenticat |
| `any` | JWT o `X-API-Key` | Flexible — suporta ambdós mètodes d'auth |
| `apikey` | `X-API-Key: sk_027_...` | Només integracions servidor a servidor |

## Format de respostes

### Èxit

Retorna els recursos directament — sense envelopa exterior:

```typescript
import { apiOk } from '@/lib/api/response'

// Recurs únic
return apiOk({ id: '123', name: 'La meva Nota' })

// Recurs creat (201)
return apiOk({ id: '456', name: 'Nova Nota' }, 201)
```

### Llistes paginades

```typescript
import { apiOk } from '@/lib/api/response'

return apiOk({
  data: elements,
  pagination: {
    page: 1,
    limit: 20,
    total: 150,
    total_pages: 8
  }
})
```

### Errors

```typescript
import { apiError } from '@/lib/api/response'

// Error bàsic
return apiError('not_found', 'Nota no trobada', 404)

// Error de validació amb detalls del camp
return apiError('validation_failed', 'L'entrada no és vàlida', 400, {
  title: 'El títol és obligatori'
})
```

### Endpoints DELETE

Retorna `204 No Content` sense body:

```typescript
import { apiOk } from '@/lib/api/response'

return new Response(null, { status: 204 })
```

### Codis d'estat HTTP

| Codi | Significat |
|---|---|
| `200` | OK — lectura o actualització reeixida |
| `201` | Created — creació de recurs reeixida |
| `204` | No Content — eliminació reeixida |
| `400` | Bad Request — entrada no vàlida |
| `401` | Unauthorized — credencials faltants o invàlides |
| `403` | Forbidden — permisos insuficients |
| `404` | Not Found — el recurs no existeix |
| `500` | Internal Server Error — error inesperat |

## Exemple complet: App de Notes

### manifest.json

```json
{
  "id": "notes",
  "name": "Notes",
  "version": "1.0.0",
  "api": true
}
```

### apps/notes/routes/route.ts (llista + crear)

```typescript
import { NextRequest } from 'next/server'
import { authenticate } from '@/lib/api/auth'
import { apiOk, apiError } from '@/lib/api/response'
import { createApiAdminClient } from '@/lib/supabase/api'
import type { UseCaseContext } from '@/lib/use-cases/types'

export async function GET(
  req: Request,
  ctx: UseCaseContext
): Promise<Response> {
  // Autenticar qualsevol usuari loguejat
  const auth = await authenticate(req, 'jwt')
  if (auth instanceof Response) return auth

  const { searchParams } = new URL(req.url)
  const page = Math.max(1, parseInt(searchParams.get('page') ?? '1', 10))
  const limit = Math.min(500, Math.max(1, parseInt(searchParams.get('limit') ?? '20', 10)))
  const offset = (page - 1) * limit

  const admin = createApiAdminClient()

  // Obtenir total de registres
  const { count } = await admin
    .from('notes')
    .select('*', { count: 'exact', head: true })
    .eq('group_id', auth.groupId)

  // Obtenir elements paginats
  const { data: notes } = await admin
    .from('notes')
    .select('id, title, content, created_at, updated_at')
    .eq('group_id', auth.groupId)
    .order('created_at', { ascending: false })
    .range(offset, offset + limit - 1)

  return apiOk({
    data: notes ?? [],
    pagination: {
      page,
      limit,
      total: count ?? 0,
      total_pages: Math.ceil((count ?? 0) / limit)
    }
  })
}

export async function POST(
  req: Request,
  ctx: UseCaseContext
): Promise<Response> {
  const auth = await authenticate(req, 'jwt')
  if (auth instanceof Response) return auth

  const body = await req.json().catch(() => null)
  if (!body || typeof body.title !== 'string') {
    return apiError('validation_failed', 'El títol és obligatori', 400, {
      title: 'Es requereix una cadena de títol vàlida'
    })
  }

  const admin = createApiAdminClient()

  const { data: note, error } = await admin
    .from('notes')
    .insert({
      title: body.title.trim(),
      content: body.content ?? '',
      group_id: auth.groupId,
      created_by: auth.userId
    })
    .select('id, title, content, created_at, updated_at')
    .single()

  if (error) {
    return apiError('server_error', 'Error en crear la nota', 500)
  }

  return apiOk(note, 201)
}
```

### apps/notes/routes/[id]/route.ts (veure, actualitzar, eliminar)

```typescript
import { authenticate } from '@/lib/api/auth'
import { apiOk, apiError } from '@/lib/api/response'
import { createApiAdminClient } from '@/lib/supabase/api'
import type { UseCaseContext } from '@/lib/use-cases/types'

export async function GET(
  req: Request,
  ctx: UseCaseContext
): Promise<Response> {
  const auth = await authenticate(req, 'jwt')
  if (auth instanceof Response) return auth

  const id = ctx.params?.id
  if (!id) return apiError('not_found', 'Nota no trobada', 404)

  const admin = createApiAdminClient()
  const { data: note } = await admin
    .from('notes')
    .select('id, title, content, created_at, updated_at')
    .eq('id', id)
    .eq('group_id', auth.groupId)
    .single()

  if (!note) return apiError('not_found', 'Nota no trobada', 404)

  return apiOk(note)
}

export async function PUT(
  req: Request,
  ctx: UseCaseContext
): Promise<Response> {
  const auth = await authenticate(req, 'jwt')
  if (auth instanceof Response) return auth

  const id = ctx.params?.id
  if (!id) return apiError('not_found', 'Nota no trobada', 404)

  const body = await req.json().catch(() => null)
  if (!body || typeof body.title !== 'string') {
    return apiError('validation_failed', 'El títol és obligatori', 400, {
      title: 'Es requereix una cadena de títol vàlida'
    })
  }

  const admin = createApiAdminClient()

  // Verificar propietat
  const { data: existing } = await admin
    .from('notes')
    .select('id')
    .eq('id', id)
    .eq('group_id', auth.groupId)
    .single()

  if (!existing) return apiError('not_found', 'Nota no trobada', 404)

  const { data: note, error } = await admin
    .from('notes')
    .update({
      title: body.title.trim(),
      content: body.content ?? '',
      updated_at: new Date().toISOString()
    })
    .eq('id', id)
    .select('id, title, content, created_at, updated_at')
    .single()

  if (error) {
    return apiError('server_error', 'Error en actualitzar la nota', 500)
  }

  return apiOk(note)
}

export async function DELETE(
  req: Request,
  ctx: UseCaseContext
): Promise<Response> {
  const auth = await authenticate(req, 'jwt')
  if (auth instanceof Response) return auth

  const id = ctx.params?.id
  if (!id) return apiError('not_found', 'Nota no trobada', 404)

  const admin = createApiAdminClient()

  const { error } = await admin
    .from('notes')
    .delete()
    .eq('id', id)
    .eq('group_id', auth.groupId)

  if (error) {
    return apiError('server_error', 'Error en eliminar la nota', 500)
  }

  return new Response(null, { status: 204 })
}
```

## Convencions de middleware

### Helpers d'autenticació

```typescript
import { authenticate } from '@/lib/api/auth'
import { apiOk, apiError } from '@/lib/api/response'
```

- **`authenticate(req, nivell)`** retorna `UseCaseContext | Response`. Verifica amb `instanceof Response` per gestionar errors d'auth.
- Per endpoints només d'admin, verifica `ctx.role === 'admin'` després de l'autenticació.

### Accés a la base de dades

```typescript
import { createApiAdminClient } from '@/lib/supabase/api'

const admin = createApiAdminClient()
// Usar per a taules específiques de l'app no incloses en tipus generats
```

### Paginació

Implementa sempre la paginació per endpoints de llista:

- `page`: nombre (default `1`, mínim `1`)
- `limit`: nombre (default `20`, mínim `1`, màxim `500`)

### Gestió d'errors

```typescript
// Retorna immediatament en errors
const auth = await authenticate(req, 'jwt')
if (auth instanceof Response) return auth

// Sempre passa un codi, missatge i estat HTTP
return apiError('not_found', 'Recurs no trobat', 404)
```
