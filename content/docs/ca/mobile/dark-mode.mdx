## Resum

L'app mòbil de 027Apps és compatible tant amb el mode clar com amb el mode fosc. L'app segueix automàticament la preferència del sistema del teu dispositiu.

## Com Funciona

1. L'app llegeix la configuració d'esquema de color del teu dispositiu (clar/fosc)
2. Tots els components del Design System s'adapten automàticament
3. Els colors de la capçalera i la barra de pestanyes s'ajusten en conseqüència

## Canviar la Configuració

**iOS**: Configuració → Pantalla i brillantor → Fosc

**Android**: Configuració → Pantalla → Tema fosc

L'app s'actualitza immediatament — no cal reiniciar-la.

## App Web

L'app web també és compatible amb el mode fosc. Activa'l des del menú de perfil a la cantonada superior dreta. La teva preferència es desa al teu perfil.
