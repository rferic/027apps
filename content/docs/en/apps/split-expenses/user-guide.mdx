## Getting Started

Split Expenses helps you track who paid for what and who owes whom. Create expense groups for different contexts (home, trip, dinner club) and add members to share costs.

### Creating a Group

1. Click **New Group**
2. Choose a name and emoji (e.g. 🏠 Home, ✈️ Trip to Paris)
3. Select the currency (EUR, USD, GBP, MXN)
4. Click Create

### Adding Members

1. Open your group
2. Go to the **Settings** tab
3. Tap **Add Member** and select from the list
4. Only group members can have expenses

### Recording an Expense

1. Tap the **+** button
2. Enter the amount and description
3. Select **who paid**
4. Choose **split type**:
   - **Equal split**: Amount divided equally among selected participants
   - **Custom split**: Set specific amounts for each person
5. Select **participants** (who shares this expense)
6. Optionally add tags (e.g. Food, Transport)
7. Tap Save

### Viewing Balances

The **Balances** tab shows:
- **What you owe**: Total you need to pay others
- **What you're owed**: Total others owe you
- **Per-person breakdown**: How much each person owes/is owed
- **Settlements**: Suggested payments to clear all debts (optimized for minimum transfers)

### Transfers

Record a payment between members:
1. Go to the **Transfers** section
2. Select who paid and who received
3. Enter the amount
4. Optionally add a note
5. Tap Save

Transfers are different from expenses — they don't split among multiple people, they're direct payments.

### Settling Debts

When someone pays what they owe:
1. View suggested settlements in the Balances tab
2. Tap a suggested settlement to create it
3. Or go to Transfers and create a manual one

### Stats

The **Stats** tab shows:
- Total group spending
- Per-person spending breakdown
- Category distribution (if using tags)
- Time-based trends

### Notifications

Group members receive push notifications and emails when:
- A new expense is added
- A settlement is created
- Someone settles a debt with you

### Tips

- Create separate groups for different contexts (don't mix home expenses with trip expenses)
- Use tags consistently to get meaningful stats
- Settle debts periodically rather than after every expense
- The "Suggested Settlements" algorithm minimizes the number of transfers needed
