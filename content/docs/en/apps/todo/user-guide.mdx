## Getting Started

The To-Do app helps you track tasks for your group. Anyone in the group can create tasks, mark them complete, or assign them to others.

### Creating a Task

1. Click the **+** button or type in the input field
2. Enter the task description
3. If you want to assign it, select a group member from the dropdown
4. Press Enter or click Add

### Managing Tasks

- **Complete**: Click the checkbox next to a task to mark it done
- **Uncomplete**: Click the checkbox again to reopen
- **Delete**: Click the trash icon to remove a task
- **Assign/Reassign**: Click the user avatar to change who's responsible
- **Filter**: Use the tab bar (All / Pending / Done) to filter tasks
- **Sort**: Tasks are sorted by creation date (newest first)

### Task Status

Each task has a status that tracks its lifecycle:

| Status | Meaning |
|--------|---------|
| **Pending** | Newly created, not yet started |
| **In Progress** | Someone is working on it |
| **Done** | Completed successfully |
| **Cancelled** | No longer needed |

Change the status by clicking the current status label and selecting a new one.

### Notifications

You'll receive an email when:
- Someone assigns a task to you
- A task assigned to you changes status

You can adjust notification preferences in the admin panel under Settings > Notifications.

### Tips

- Use descriptive task names — "Fix kitchen sink" is better than "Fix"
- Assign tasks to specific people to avoid the bystander effect
- Mark tasks in progress when you start working on them so others know
- Use status changes (not just complete/delete) to communicate progress
