## Getting Started

Inspiration is your group's idea board. Propose improvements, report issues, request features, or submit any kind of idea. Other members can vote and comment.

### Submitting an Idea

1. Click **New Request**
2. Choose a type:
   - **Idea** — A suggestion or improvement
   - **Feature** — A new feature request
   - **Bug** — Something that's broken
   - **New App** — Propose adding a new application
   - **Improvement** — Enhance an existing feature
   - **Other** — Anything else
3. Write a title and description
4. Select which app it relates to (optional)
5. Click Submit

### Voting

Vote on ideas to show support:
- Click the **thumbs up** button on any idea
- Click again to remove your vote
- Vote count shows how many people support the idea
- Higher-voted ideas appear at the top

### Comments

Discuss ideas in the comment section:
- Click on an idea to open its detail page
- Scroll to the Comments section
- Write your comment and click Post
- Comments appear in chronological order

### Status Tracking

Ideas go through status stages:

| Status | Meaning |
|--------|---------|
| **Pending** | Submitted, awaiting review |
| **Reviewing** | Being evaluated by admins |
| **Approved** | Accepted, will be implemented |
| **In Progress** | Currently being worked on |
| **Completed** | Done and deployed |
| **Rejected** | Not feasible or not a priority |
| **Duplicate** | Already exists as another idea |

Status changes are managed by group admins.

### Notifications

You'll receive email notifications when:
- Someone comments on your idea
- An idea you voted on changes status
- A new idea is submitted to the group (admins only)

### Tips

- Search existing ideas before submitting — it might already be proposed
- Write clear, specific titles
- Add relevant context in the description
- Vote regularly to help prioritize what matters
