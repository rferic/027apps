# Admin — Split Expenses

## List all expense groups

```bash
GET /api/v1/admin/apps/split-expenses
```

Returns all expense groups across all groups with member counts and total amounts. Requires `admin` role.

**Auth:** JWT (`admin` role)

**Response 200:**
```json
[
  {
    "id": "uuid",
    "group_id": "uuid",
    "title": "Home Expenses",
    "emoji": "🏠",
    "currency": "EUR",
    "created_by": "uuid",
    "created_at": "2026-01-15T10:30:00Z",
    "member_count": 4,
    "total_amount": 1250.50
  }
]
```

## Global statistics

```bash
GET /api/v1/admin/apps/split-expenses/stats
```

Returns aggregate statistics across all groups. Requires `admin` role.

**Auth:** JWT (`admin` role)

**Response 200:**
```json
{
  "total_groups": 3,
  "total_expenses": 156,
  "total_amount": 4520.75,
  "total_members": 12,
  "by_currency": {
    "EUR": { "groups": 2, "total": 3200.00 },
    "USD": { "groups": 1, "total": 1320.75 }
  }
}
```
