Aufgaben ist ein einfacher Aufgabenverwalter für Gruppen. Jeder Benutzer kann seine eigenen Aufgaben erstellen, erledigen und löschen. Administratoren können Limits pro Benutzer festlegen.

## Funktionen

- **Aufgaben erstellen**: Füge Aufgaben mit einem Titel hinzu
- **Aufgaben erledigen**: Markiere Aufgaben mit einem Klick als erledigt
- **Benutzer-Limits**: Admins können maximale Aufgaben pro Benutzer festlegen
- **Sichtbarkeit**: Aufgaben können öffentlich oder privat sein
- **Widget**: Dashboard-Widget mit ausstehenden und erledigten Aufgaben

## Configuration

Admins can configure the maximum number of tasks per user in the admin panel.

| Key | Type | Default | Min | Max |
|-----|------|---------|-----|-----|
| `max_items` | number | 50 | 1 | 1000 |
