# 027Apps Docs

Plattformdokumentation für Entwickler, die mit 027Apps arbeiten oder es in ihre Projekte integrieren.

027Apps ist eine Open-Source-Plattform für Gruppen-Apps — für Familien, Teams oder jedes beliebige Kollektiv. Jede Gruppe verwaltet ihre eigenen Benutzer, Rollen und App-Berechtigungen.

## Erste Schritte

Die Plattform stellt eine REST API für Authentifizierung, Benutzerverwaltung, App-Zugriff und Sprachkonfiguration bereit. Alle Endpoints sind unter `/api/v1/` versioniert.

Beginnen Sie mit [Authentifizierung](/doc/api/authentication), um ein Sitzungstoken zu erhalten, und erkunden Sie dann die verfügbaren Endpoints.
