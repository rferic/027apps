To-Do is a straightforward task manager for groups. Each user can create, complete, and delete their own tasks. Supports per-user limits configured by admins.

## Features

- **Create tasks**: Add tasks with a title
- **Complete tasks**: Mark tasks as done with a single click
- **Per-user limits**: Admins can set max items per user
- **Visibility**: Tasks can be public or private
- **Widget**: Dashboard widget showing pending and completed counts

## Configuration

Admins can configure the maximum number of tasks per user in the admin panel.

| Key | Type | Default | Min | Max |
|-----|------|---------|-----|-----|
| `max_items` | number | 50 | 1 | 1000 |
