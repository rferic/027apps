Da fare è un semplice gestore di attività per gruppi. Ogni utente può creare, completare ed eliminare le proprie attività. Gli amministratori possono impostare limiti per utente.

## Funzionalità

- **Creare attività**: Aggiungi attività con un titolo
- **Completare attività**: Segna le attività come completate con un clic
- **Limiti per utente**: Gli admin possono impostare un massimo di attività per utente
- **Visibilità**: Le attività possono essere pubbliche o private
- **Widget**: Widget nella dashboard con attività in sospeso e completate

## Configuration

Admins can configure the maximum number of tasks per user in the admin panel.

| Key | Type | Default | Min | Max |
|-----|------|---------|-----|-----|
| `max_items` | number | 50 | 1 | 1000 |
