# Admin — Split Expenses

## Lister tous les groupes de dépenses

```bash
GET /api/v1/admin/apps/split-expenses
```

Retourne tous les groupes de dépenses avec le nombre de membres et les montants totaux. Nécessite le rôle `admin`.

**Auth:** JWT (`admin` role)

**Réponse 200:**
```json
[
  {
    "id": "uuid",
    "group_id": "uuid",
    "title": "Home Expenses",
    "emoji": "🏠",
    "currency": "EUR",
    "created_by": "uuid",
    "created_at": "2026-01-15T10:30:00Z",
    "member_count": 4,
    "total_amount": 1250.50
  }
]
```

## Statistiques globales

```bash
GET /api/v1/admin/apps/split-expenses/stats
```

Retourne les statistiques agrégées de tous les groupes. Nécessite le rôle `admin`.

**Auth:** JWT (`admin` role)

**Réponse 200:**
```json
{
  "total_groups": 3,
  "total_expenses": 156,
  "total_amount": 4520.75,
  "total_members": 12,
  "by_currency": {
    "EUR": { "groups": 2, "total": 3200.00 },
    "USD": { "groups": 1, "total": 1320.75 }
  }
}
```
