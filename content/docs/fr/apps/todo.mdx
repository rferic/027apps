À faire est un gestionnaire de tâches simple pour les groupes. Chaque utilisateur peut créer, compléter et supprimer ses propres tâches. Les administrateurs peuvent définir des limites par utilisateur.

## Fonctionnalités

- **Créer des tâches**: Ajoutez des tâches avec un titre
- **Compléter des tâches**: Marquez les tâches comme terminées en un clic
- **Limites par utilisateur**: Les admins peuvent définir un maximum de tâches par utilisateur
- **Visibilité**: Les tâches peuvent être publiques ou privées
- **Widget**: Widget tableau de bord avec les tâches en attente et terminées

## Configuration

Admins can configure the maximum number of tasks per user in the admin panel.

| Key | Type | Default | Min | Max |
|-----|------|---------|-----|-----|
| `max_items` | number | 50 | 1 | 1000 |
