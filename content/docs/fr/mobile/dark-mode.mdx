## Aperçu

L'application mobile 027Apps prend en charge les modes clair et sombre. L'application suit automatiquement les préférences système de votre appareil.

## Comment Ça Marche

1. L'application lit le paramètre de thème de couleur de votre appareil (clair/sombre)
2. Tous les composants du Design System s'adaptent automatiquement
3. Les couleurs de l'en-tête et de la barre d'onglets s'ajustent en conséquence

## Modifier le Paramètre

**iOS** : Réglages → Luminosité et affichage → Sombre

**Android** : Paramètres → Affichage → Thème sombre

L'application se met à jour immédiatement — aucun redémarrage nécessaire.

## Application Web

L'application web prend également en charge le mode sombre. Activez-le depuis le menu du profil dans le coin supérieur droit. Votre préférence est enregistrée dans votre profil.
