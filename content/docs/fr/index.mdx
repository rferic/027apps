# 027Apps Docs

Documentation de la plateforme pour les développeurs qui construisent avec 027Apps ou l'intègrent dans leurs projets.

027Apps est une plateforme open-source pour les applications de groupe — pour les familles, les équipes ou tout collectif. Chaque groupe gère ses propres utilisateurs, rôles et permissions d'application.

## Premiers pas

La plateforme expose une API REST pour l'authentification, la gestion des utilisateurs, l'accès aux applications et la configuration des langues. Tous les endpoints sont versionnés sous `/api/v1/`.

Commencez par [Authentification](/doc/api/authentication) pour obtenir un token de session, puis explorez les endpoints disponibles.
