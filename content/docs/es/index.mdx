# 027Apps Docs

Documentación de la plataforma para desarrolladores que construyen con 027Apps o lo integran en sus proyectos.

027Apps es una plataforma de código abierto para aplicaciones de grupo — para familias, equipos o cualquier colectivo. Cada grupo gestiona sus propios usuarios, roles y permisos de aplicación.

## Primeros pasos

La plataforma expone una API REST para autenticación, gestión de usuarios, acceso a aplicaciones y configuración de idiomas. Todos los endpoints están versionados bajo `/api/v1/`.

Comienza con [Autenticación](/doc/api/authentication) para obtener un token de sesión y luego explora los endpoints disponibles.
