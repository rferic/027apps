To-Do es un gestor de tareas sencillo para grupos. Cada usuario puede crear, completar y eliminar sus propias tareas. Los administradores pueden configurar límites por usuario.

## Funcionalidades

- **Crear tareas**: Añade tareas con un título
- **Completar tareas**: Marca tareas como hechas con un solo clic
- **Límites por usuario**: Los administradores pueden establecer un máximo de tareas por usuario
- **Visibilidad**: Las tareas pueden ser públicas o privadas
- **Widget**: Widget en el panel que muestra tareas pendientes y completadas

## Configuration

Admins can configure the maximum number of tasks per user in the admin panel.

| Key | Type | Default | Min | Max |
|-----|------|---------|-----|-----|
| `max_items` | number | 50 | 1 | 1000 |
