# To-Do App

A full-featured task management app for groups. Tasks can be public (visible to all group members) or private (visible only to the creator). Supports categories, priorities, due dates, assignments, and email notifications.

## Prerequisites

- Platform version ≥ 1.0.0
- No dependencies on other apps

## Installation

Install from the admin panel at `/admin/apps`. The system will:

1. Create the `todo_items`, `todo_categories`, and `todo_notification_prefs` tables with RLS policies
2. Enable per-user row isolation automatically

## Features

- **Public/Private tasks** — Public tasks are visible to all group members; private tasks are only visible to the creator
- **Assignment** — Public tasks can be assigned to any group member. Tasks are unassigned when a member leaves the group
- **Categories** — Global categories with emoji + color, managed by admins
- **Priorities** — `urgent`, `high`, `medium`, `low` with visual indicators
- **Due dates** — Optional due dates with overdue highlighting
- **Email notifications** — Configurable per-user: on assignment, on status change
- **Dashboard widgets** — "My tasks" and "Group tasks" widgets

## Views

| View | URL | Description |
|------|-----|-------------|
| Public | `/apps/todo` | Full task list with tabs, filters, create/edit/delete |
| Admin | `/admin/apps/todo` | Category management + all tasks overview |
| Widget | Dashboard | My tasks & unassigned group tasks with "Take" button |

## Uninstallation

Uninstalling removes all TODO tables and associated data permanently.
